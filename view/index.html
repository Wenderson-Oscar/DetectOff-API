Bem-Vindos à API de Detecção de Comentários Ofensivos

Sobre a API

Esta API é usada para prever se um comentário é ofensivo ou não.

Endpoints:

Dados Ofensivos
Dados Não Ofensivos

Exemplos de Comentários

Comentário Ofensivo

"Esse site é horrível, não serve pra nada."

Resultado: Ofensivo

Comentário Não Ofensivo

"Parabéns pelo ótimo trabalho!"

Resultado: Não Ofensivo
